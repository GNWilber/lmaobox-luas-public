# WilCrit
Vanilla's "CritBar" lacks detailed information, so I created a cleaner alternative.
This mod shows no indicator when a crit is fully ready, but if you need extra damage or more shots, it precisely displays exactly what is required.

The position and state of the layout are fully adjustable in real-time using the included settings GUI, rather than having to manually edit the source code.

*Requires `wilgui.lua` to display the settings window.*
